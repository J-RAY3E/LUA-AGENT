# math.sqrt

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua
math.sqrt(x)
```

## Description
Returns the square root of x. (You can also use the expression x^0.5 to compute this value.)

## Parameters
_None_

## Returns
- (lua_Number): The square root of x.
