# math.fmod

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua
math.fmod(x, y)
```

## Description
Returns the remainder of the division of `x` by `y` that rounds the quotient towards zero. (integer/float)

## Parameters
_None_

## Returns
- (number): The remainder of the division of `x` by `y` that rounds the quotient towards zero. (integer/float)
