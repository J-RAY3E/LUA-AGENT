# math.ult

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua
math.ult(m, n)
```

## Description
Returns a boolean, true if and only if integer m is below integer n when they are compared as unsigned integers.

## Parameters
_None_

## Returns
- (boolean): true if and only if integer m is below integer n when they are compared as unsigned integers.
