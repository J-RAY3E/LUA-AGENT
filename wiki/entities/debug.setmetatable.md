# debug.setmetatable

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua
value, table
```

## Description
Sets the metatable for the given `value` to the given `table` (which can be **nil**). Returns `value`.

## Parameters
_None_

## Returns
- (value): The original value
