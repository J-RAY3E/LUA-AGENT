# math.modf

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua
math.modf(x)
```

## Description
Returns the integral part of `x` and the fractional part of `x`. Its second result is always a float.

## Parameters
_None_

## Returns
- (number): The integral part of `x`.
- (number): The fractional part of `x`, always a float.

## Implementation Code
```c
math.modf(x)
```
