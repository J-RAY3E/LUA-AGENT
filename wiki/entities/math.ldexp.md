# math.ldexp

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua
math.ldexp(m, e)
```

## Description
Returns m2e, where e is an integer.

## Parameters
_None_

## Returns
- (number): m2e, where e is an integer.

## Implementation Code
```c
return m * 2^e
```
