# string.gsub

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua
string.gsub(string s, string pattern, string repl, integer n)
```

## Description
Global SUBstitution

## Parameters
- `s` (string): The string to be modified.
- `n` (integer): The number of times to replace the pattern.
- `pattern` (string): The pattern to search for.
- `repl` (string): The replacement string or table.

## Returns
- (string): The modified string.
