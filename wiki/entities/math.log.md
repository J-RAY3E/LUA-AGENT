# math.log

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua
math.log(x [, base])
```

## Description
Returns the logarithm of `x` in the given base. The default for `base` is *e* (so that the function returns the natural logarithm of `x`).

## Parameters
_None_

## Returns
- (number): The logarithm of `x` in the given base.
