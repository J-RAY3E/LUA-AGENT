# math.asin

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua

```

## Description
Returns the arc sine of `x` (in radians).

## Parameters
_None_

## Returns
- (number): The arc sine of `x` (in radians).
