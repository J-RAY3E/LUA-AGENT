# math.type

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua
math.type(x)
```

## Description
Returns "integer" if x is an integer, "float" if it is a float, or fail if x is not a number.

## Parameters
_None_

## Returns
- (string): Returns "integer" if x is an integer, "float" if it is a float, or fail if x is not a number.
