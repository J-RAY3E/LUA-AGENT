# math.max

**Type**: Function  
**Module**: [[Standard Libraries]]  

## Signature
```lua
math.max(x, ...) -> number
```

## Description
Returns the argument with the maximum value, according to the Lua operator '<'.

## Parameters
_None_

## Returns
- (number): The maximum value among the arguments.

## Implementation Code
```c
return math.max(...)
```
